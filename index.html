SRKey v1.3.0

A free, browser-based tool that generates PICOT-based search strategies for systematic reviews using Claude AI — no API key or installation required.

PICOT framework

ElementStands forExample
PPopulationAdults with type 2 diabetes
IInterventionSGLT-2 inhibitors
CComparatorPlacebo or standard care
OOutcomeCardiovascular events, HbA1c
TTime / Study typeRCTs, inception–2024

Features

Search term types

  • MeSH + Title/Abstract — NLM MeSH headings combined with free-text keywords, synonyms, and abbreviations (recommended)
  • MeSH only — controlled vocabulary terms only
  • Title/Abstract only — free-text keywords and synonyms only

Study type filters

  • Randomised Controlled Trials (RCTs)
  • Observational studies (cohort, case-control, cross-sectional)
  • Systematic reviews / Meta-analyses
  • All study types (no filter applied)

Multiple study types can be selected simultaneously.

Database support

DatabaseQuery syntax used
PubMed[MeSH Terms], [tiab]
Embase/exp, :ti,ab
Cochrane CENTRALMeSH + free-text
ScopusTITLE-ABS-KEY()
Web of ScienceTS=()

Select one or more databases — all queries are generated and displayed simultaneously.

Date range

Leave unchecked to search from inception to today. Enable the date picker to restrict by publication year. Date filter syntax is applied automatically per database.

Export to Word

Downloads a formatted .docx file containing the PICOT framework table, all database queries, and study metadata (objective, date, term type, study type, date range).

Tips for best results

  • Be specific in your objective — include population, intervention, and comparator where possible
  • Always verify MeSH terms at meshb.nlm.nih.gov before use
  • Generated queries are a starting point — additional terms may be needed based on your protocol
  • Validate your final strategy with a medical librarian before conducting the review
